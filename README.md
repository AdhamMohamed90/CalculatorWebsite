# 🧮 Simple Calculator

A simple, responsive, and user-friendly calculator built with **HTML5**, **CSS3**, and **JavaScript**. It supports basic arithmetic operations and provides a clean interface with smooth hover effects and responsive design.

## ✨ Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 🧹 Clear Function
* 📱 Responsive Design
* 🎨 Hover Effects & Smooth Transitions

## 🛠️ Technologies Used

* **HTML5**
* **CSS3**

  * Grid Layout
  * Flexbox
  * Transitions
* **JavaScript**

## 🚀 How to Use

1. Click the number buttons to enter values.
2. Select an operator (**+**, **-**, **×**, or **÷**).
3. Press **=** to calculate the result.
4. Press **C** to clear the calculator display.
